
# 📌 ATM Simulation in Java

## 📖 Project Description
This project is a simple *ATM Simulation system* developed in Java. It demonstrates basic banking operations like PIN authentication, balance checking, and cash withdrawal.

The main goal of this project is to understand how real-world ATM systems work while practicing Object-Oriented Programming (OOP) concepts such as Encapsulation.

---

## 🎯 Why This Project?
This project was created to:
- Understand real-life application of Java programming  
- Practice OOP concepts (especially Encapsulation)  
- Learn how ATM machines function internally  
- Improve logical thinking and problem-solving skills  

---

## 🚀 Features / Utility
- Secure PIN-based authentication  
- Withdraw money with proper validation  
- Check account balance instantly  
- Handles invalid input using exception handling  
- Simple menu-driven interface  

---

## 🛠️ Technologies Used
- Java  
- OOP Concepts  
- Scanner Class  

---

## 📂 Project Structure
ATMSimulation.java

├── Account Class  
│   ├── withdraw()  
│   └── getBalance()  

├── ATM Class  
│   └── authenticate()  

└── Main Class (ATMSimulation)  
    └── main()  

---

## ▶️ How to Run
1. Open project in VS Code or any Java IDE  
2. Compile the program:
   javac ATMSimulation.java  

3. Run the program:
   java ATMSimulation

## ⚠️ Validation Rules
- PIN must be correct (1234)  
- Withdrawal amount must be:
  - Greater than 0  
  - Less than or equal to available balance
