# 🏧 Smart ATM Simulation System

A console-based ATM Simulation System developed in Java using Object-Oriented Programming principles. This project replicates real-world ATM operations and provides a simple banking experience through the command line.

---

## ✨ Key Features

🔐 Secure PIN Authentication

💰 Cash Withdrawal

📥 Money Deposit

📊 Balance Inquiry

🔄 PIN Change Facility

🚪 Safe Exit Option

---

## 🛠 Technologies Used

- Java
- OOP Concepts
  - Encapsulation
  - Inheritance
  - Polymorphism
  - Abstraction
- Scanner Class

---

## 📋 Workflow

1. User enters ATM PIN
2. System verifies credentials
3. ATM menu is displayed
4. User selects an operation
5. Transaction is processed instantly

---

## 📸 Sample Execution

```text
Enter ATM PIN: 1234

===== ATM MENU =====
1. Withdraw Money
2. Deposit Money
3. Check Balance
4. Change PIN
5. Exit

Enter Choice: 3

Current Balance: ₹5000.00
```
